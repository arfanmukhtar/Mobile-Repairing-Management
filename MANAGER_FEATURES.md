# Manager Features

## Dashboard
- **Shop Overview** - View stats for assigned shop only
- **Active Repairs** - See repairs not yet collected
- **Today's Stats** - Today's bookings, payments, and revenue
- **Recent Transactions** - Latest 5 paid repairs

## Repair Bookings
- **Create Bookings** - Add new repair bookings for customers
- **Select Category** - Choose device category and subcategory
- **Select Service** - Choose repair service type
- **Set Attributes** - Define device details (color, storage, condition)
- **Set Pricing** - Enter repair price and advance payment
- **Edit Bookings** - Update booking details
- **Delete Bookings** - Remove bookings
- **Update Status** - Change repair status (Booked, In Progress, Fixed, etc.)
- **Update Payment** - Mark as paid, set payment type (Cash/Card)
- **Manage Parts** - Add/remove parts used in repair
- **Auto Stock** - Parts automatically decrease stock when added

## Kanban Board
- **Visual Status Board** - Drag-and-drop repair cards by status
- **Status Columns** - Booked, In Progress, In Lab, Fixed, Not Fixable, Customer Collected
- **Drag to Update** - Move cards to change status
- **Quick View** - See customer, category, price on each card
- **Filter by Status** - Show/hide specific status columns

## Customers
- **Create Customers** - Add customer records
- **Customer Details** - Name, email, phone, address
- **Edit Customers** - Update customer information
- **Delete Customers** - Remove customer records
- **Search Customers** - Find customers quickly

## Categories
- **View Categories** - See all device categories and subcategories
- **Cannot Edit** - Categories managed by owner only

## Inventory - Parts
- **View Parts** - See global parts and shop-specific parts
- **Cannot Create** - Parts managed by owner only

## Inventory - Stock
- **View Stock** - See stock levels for shop
- **Add Stock** - Add parts inventory by category
- **Bulk Entry Grid** - Add multiple parts at once
- **Filter by Category** - View stock by device type
- **Cannot Transfer** - Stock management limited to own shop

## Services
- **View Services** - See all available repair services
- **Cannot Edit** - Services managed by owner only

## Email Templates
- **View Owner Templates** - See all templates created by owner
- **Customize Templates** - Create shop-specific versions
- **Reset to Default** - Remove customization, use owner's template
- **Cannot Create New** - Base templates created by owner only

## Reports - Repairs
- **Time Comparisons** - Today vs Yesterday, This Week vs Last Week, etc.
- **Auto-Filtered** - Shows only own shop (no shop filter)
- **Date Range** - Filter by custom dates
- **Status Filter** - Filter by repair status
- **Detailed Listing** - View all repairs with pagination
- **Click for Details** - View full repair information

## Reports - Purchases
- **Time Comparisons** - Same time-based stats
- **Amount Tracking** - Total, paid, due amounts
- **Auto-Filtered** - Shows only own shop
- **Date Range** - Filter by custom dates
- **Purchase Listing** - View all purchases

## Reports - Parts Used
- **Usage Analysis** - See which parts are used most
- **Month Filter** - View by month (default: current month)
- **Category Filter** - Filter by device category
- **Auto-Filtered** - Shows only own shop
- **Only Used Parts** - Shows only parts with usage > 0
- **Top 5 Chart** - Visual chart of most used parts

## Settings
- **Profile Settings** - Update name, email, password
- **Shop SMTP** - Configure shop-specific email settings (optional)
- **Shop Currency** - Set shop currency (optional, inherits from owner)
- **Notification Preferences** - Configure notifications
- **Falls Back to Owner** - Uses owner's settings if not configured
