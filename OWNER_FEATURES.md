# Owner Features

## Dashboard
- **Overview Stats** - View total shops, managers, revenue, and active repairs
- **Shop Filter** - Filter dashboard data by specific shop or view all shops
- **Revenue Tracking** - Monitor today's payments, cash/card breakdown, and total transactions

## Shop Management
- **Create Shops** - Add new repair shop locations
- **Edit Shops** - Update shop details (name, address, phone, email)
- **Delete Shops** - Remove shops from system
- **View Shop Performance** - See revenue and stats per shop

## Manager Management
- **Create Managers** - Add manager accounts and assign to shops
- **Edit Managers** - Update manager information and shop assignments
- **Delete Managers** - Remove manager accounts
- **Access Control** - Managers can only access their assigned shop

## Inventory - Parts
- **Create Global Parts** - Add parts available to all shops
- **Create Shop Parts** - Add parts specific to one shop
- **Edit Parts** - Update part details and pricing
- **Delete Parts** - Remove parts from system

## Inventory - Stock
- **Add Stock** - Add parts inventory by category and shop
- **Bulk Entry Grid** - Add multiple parts at once using spreadsheet-style grid
- **View Stock Levels** - Monitor stock by category and shop
- **Advanced Filters** - Filter by category, part, shop, and stock status
- **Export Stock** - Download stock data

## Inventory - Purchases
- **Record Purchases** - Log inventory purchases with supplier details
- **Track Payments** - Monitor paid and due amounts
- **Purchase History** - View all purchase records
- **Supplier Management** - Track purchases by supplier

## Services - Categories
- **Create Categories** - Add device categories (iPhone, Samsung, etc.)
- **Create Subcategories** - Add models under categories (iPhone 13, Galaxy S21, etc.)
- **Manage Hierarchy** - Organize categories in parent-child structure
- **Edit/Delete** - Update or remove categories

## Services - Attributes
- **Create Attributes** - Define product attributes (Color, Storage, Condition)
- **Set Values** - Add attribute options (Black/White, 64GB/128GB, etc.)
- **Assign to Categories** - Link attributes to specific categories
- **Edit/Delete** - Update or remove attributes

## Services - Services
- **Create Services** - Add repair service types (Screen Replacement, Battery, etc.)
- **Set Pricing** - Define default prices for services
- **Edit/Delete** - Update or remove services
- **Available to All Shops** - All shops can use these services

## Email Templates
- **Create Templates** - Design email templates for different scenarios
- **Rich Text Editor** - Format emails with colors, images, links
- **Variable Insertion** - Add dynamic fields (customer name, booking ID, etc.)
- **Template Types** - Booking confirmation, payment received, repair completed, etc.
- **Shop Customization** - Managers can customize templates for their shop

## Reports - Repairs
- **Time Comparisons** - Today vs Yesterday, This Week vs Last Week, This Month vs Last Month, This Year vs Last Year
- **Shop Filter** - View all shops or filter by specific shop
- **Date Range** - Filter by custom date range
- **Status Filter** - Filter by repair status
- **Detailed Listing** - View all repairs with pagination
- **Click for Details** - View full repair information in modal

## Reports - Purchases
- **Time Comparisons** - Same time-based stats as repairs
- **Amount Tracking** - Total amount, paid amount, due amount
- **Shop Filter** - View all shops or specific shop
- **Date Range** - Filter by custom dates
- **Purchase Listing** - View all purchases with supplier and payment details

## Reports - Parts Used
- **Usage Analysis** - See which parts are used most frequently
- **Month Filter** - View parts usage by month (default: current month)
- **Category Filter** - Filter by device category
- **Shop Filter** - View all shops or specific shop
- **Only Used Parts** - Shows only parts with usage > 0
- **Top 5 Chart** - Visual chart of most used parts
- **Usage Percentage** - See percentage breakdown of part usage

## Settings
- **Profile Settings** - Update name, email, password
- **SMTP Configuration** - Configure email sending settings
- **Currency Settings** - Set default currency for all shops
- **Notification Preferences** - Configure email and system notifications
- **Shop Settings** - Default settings inherited by all shops
